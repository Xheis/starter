This project was created to provide a simple way to test your HTML styles.

Its main use case is for visualizing the CSS for personal sites and blogs, but its simplicity keeps it versatile. We recommend using the following tags when fully-skinning your site:

  • h1-h6
  • p, strong, em
  • ol, ul, li
  • pre > code (for code blocks)
  • hr
  • table, thead, tr, th, tbody
  • form, input, textarea
  • button, input[type=submit]
